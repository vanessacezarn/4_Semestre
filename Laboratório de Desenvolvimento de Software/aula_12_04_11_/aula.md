## API - interface de programação de aplicações
- conjunto de regras e definições que permite que diferentes software se comuniquem entre si
- define a maneira como componentes de software devem interagir
- objetivo: facilitar a integração entre sistemas e permitir que desenvolvedores acessem funcionalidades específicas de software sem precisar entender completamente sua implementação interna
- abstração de funcionalidade: APIs fornecem uma camada de abstração, permitindo que os desenvolvedores
- padrões de comunicação: APIs seguem padrões específicos de comunicação, como o uso de protocolos como HTTP/HTTPS e métodos (get, post, put, delete)
- reutilização de código:
- APIs Web: são interfaces que utilizam os protocolos HTTP/HTTPS para comunicação, elas são amplamente utilizadas para a construção de serviços online
- RESTful APIs: categoria de APIs web baseada nos princípios da arquitetura REST (Representational State Transfer)
